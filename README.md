# Sales Data Analysis Project

## Overview
This project analyzes a sales dataset to identify key business insights related to performance across cities, product categories, and time.  
The analysis focuses on revenue, profitability, and sales trends to support data-driven decision-making.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
The dataset contains sales transaction records with the following fields:
- OrderID
- Date
- City
- Category
- Product
- Sales
- Quantity
- Profit

## Analysis Performed
- Data exploration and structure analysis
- Missing values and duplicate check
- Sales and profit analysis by city
- Sales and profit analysis by category
- Identification of top-performing products
- Sales trend analysis over time
- Profit margin calculation (overall and by city)
- Data visualization using Matplotlib

## Key Insights
- Dammam generated the highest total sales and overall profit.
- Jeddah achieved the highest profit margin (~13%), indicating higher efficiency.
- Technology is the top-performing category in both sales and profit.
- Laptop is the best-selling and most profitable product.
- Sales peaked on 2024-01-14 due to a high-value transaction.
- Total Sales: 11,910  
- Total Profit: 1,430  
- Overall Profit Margin: ~12%

## Business Recommendations
- Focus on Technology products, especially laptops, as they drive the highest revenue and profit.
- Analyze operational efficiency in Jeddah and replicate successful practices in other cities.
