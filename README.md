# Sales Data Analysis Project

## Overview
This project analyzes a sales dataset to identify key business insights related to performance across cities, product categories, and time.  
The goal is to understand sales patterns, profitability, and top-performing areas to support data-driven decision-making.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset
The dataset contains sales transaction records with the following fields:
- OrderID
- Date
- City
- Category
- Product
- Sales
- Quantity
- Profit

## Analysis Performed
- Sales and profit analysis by city
- Sales and profit analysis by category
- Identification of top-performing products
- Sales trend analysis over time
- Overall and city-level profit margin analysis

## Key Insights
- Dammam generated the highest total sales and profit.
- Jeddah achieved the highest profit margin (~13%), indicating higher operational efficiency.
- Technology is the top-performing category in both sales and profit.
- Laptop is the best-selling and most profitable product.
- Total Sales: 11,910  
- Total Profit: 1,430  
- Overall Profit Margin: ~12%
